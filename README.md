## Getting Started

### Installation

1. **Clone the Repository:**

   ```bash
   git clone git@github.com:skyfire-xyz/sky-crawler.git
   cd sky-crawler
   ```

2. **Install Dependencies:**

   ```bash
   npm install crawlee puppeteer
   npm install typescript ts-node @types/node
   ```

3. **Start the Development Server:**
   ```bash
   npm start
   ```
